# 不动产数据生成算子

## 功能说明

本算子用于生成不动产信息查询结果数据，包含以下信息：

- 姓名（证件号码）
- 查询目的
- 电脑查册人
- 登记字号
- 身份证号
- 地址
- 产权证号
- 证明流水号
- 打印日期
- 表格数据（可选）

## 使用场景

- 测试数据处理流程
- 生成模拟数据用于演示
- 批量生成测试样本

## 参数说明

| 参数             | 类型     | 默认值 | 说明                             |
| ---------------- | -------- | ------ | -------------------------------- |
| dataCount        | 数字     | 10     | 生成数据组数（1-100）            |
| outputFormat     | 下拉选择 | json   | 输出格式：JSON格式 或 纯文本格式 |
| includeTableData | 开关     | true   | 是否在输出中包含详细的表格数据   |
