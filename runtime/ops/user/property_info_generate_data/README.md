# 不动产数据生成算子

## 功能说明

本算子用于生成不动产信息查询结果数据，包含以下信息：

- 姓名（证件号码）
- 查询目的
- 电脑查册人
- 证明流水号
- 打印日期
- 表格数据（包含状态、合同号、地址、产权证号、规划用途、建筑面积、数据来源）

## 输入参数

| 参数名 | 类型        | 默认值 | 说明                        |
| ------ | ----------- | ------ | --------------------------- |
| count  | inputNumber | 1      | 生成不动产数据组数（1-100） |

## 输出格式

算子输出为 JSON 格式的文本，包含多组不动产查询结果数据。

### 示例输出

```json
[
    {
        "index": 1,
        "省份": "东华省",
        "姓名（证件号码）": "王伟（110101199001011234）",
        "查询目的": "担保用",
        "电脑查册人": "ABCD-EFGH",
        "证明流水号": "20250115123456",
        "打印日期": "2025年01月15日 14:30:25",
        "表格数据": [
            {
                "序号": 1,
                "状态": "正常",
                "合同号": "123456789012345",
                "地址": "东华省东华市人民路小区5栋2单元1203室",
                "产权证号": "东华12345",
                "规划用途": "住宅",
                "建筑面积": "120.5㎡",
                "数据来源": "东华市不动产登记中心"
            }
        ]
    }
]
```

## 注意事项

1. 本算子使用编造的省份和城市名称，避免使用真实地理位置信息
2. 身份证号格式符合18位身份证校验规则，但为随机生成，非真实数据
3. 每组数据包含1-2条不动产记录
4. 打印日期范围为2025年全年

## 使用场景

适用于需要生成不动产查询结果测试数据的场景，如：

- 数据标注训练
- 系统功能测试
- 文档生成流程测试
