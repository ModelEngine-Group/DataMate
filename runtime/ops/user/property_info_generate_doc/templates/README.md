# Word 模板文件

此目录应包含 Word 模板文件：`不动产信息查询结果模板.docx`

## 模板要求

Word 模板必须包含以下占位符：

### 段落占位符

- `{{省份}}` - 省份名称
- `{{姓名（证件号码）}}` - 姓名和身份证号
- `{{姓名}}` - 仅姓名
- `{{查询目的}}` - 查询目的
- `{{电脑查册人}}` - 电脑查册人编号
- `{{证明流水号}}` - 证明流水号
- `{{打印日期}}` - 打印日期

### 表格占位符

- `{{序号}}` - 序号
- `{{状态}}` - 状态
- `{{合同号}}` - 合同号
- `{{地址}}` - 地址
- `{{产权证号}}` - 产权证号
- `{{规划用途}}` - 规划用途
- `{{建筑面积}}` - 建筑面积
- `{{数据来源}}` - 数据来源

## 注意事项

1. 请将实际的 Word 模板文件命名为 `不动产信息查询结果模板.docx` 并放置在此目录中
2. 模板中必须包含至少一个包含 `{{序号}}` 占位符的表格，用于动态生成多行数据
3. 标题段落（包含"不动产信息查询结果"）会被自动加粗

## 模板示例结构

```
标题：{{省份}}不动产信息查询结果

姓名（证件号码）：{{姓名（证件号码）}}
查询目的：{{查询目的}}
电脑查册人：{{电脑查册人}}
证明流水号：{{证明流水号}}
打印日期：{{打印日期}}

表格1（表头加粗）：
| 姓名（证件号码） | 查询目的 | ... |

表格2：
| ... |

表格3（所有单元格加粗）：
| ... |

表格4（包含 {{序号}} 占位符，用于动态生成多行）：
| 序号 | 状态 | 合同号 | 地址 | ... |
| {{序号}} | {{状态}} | {{合同号}} | {{地址}} | ... |
```
